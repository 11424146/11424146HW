## Topic 2 (Short Introduction)

Topic: Terminal Velocity of a Parachute  
Main idea: Speed goes up fast first, then slows down, and finally becomes almost constant.

Main equation:

$$
m\frac{dv}{dt} = mg - kv
$$

Solution:

$$
v(t)=\frac{mg}{k}\left(1-e^{-\frac{k}{m}t}\right)
$$

Terminal velocity:

$$
v_T=\frac{mg}{k}
$$
